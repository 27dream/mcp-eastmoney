# 📝 Sample Prompts — 50+ 条实战 prompts 给 Claude / Cursor

覆盖：行情查询、主力资金分析、板块轮动、K 线技术分析、复合策略选股。

## 1. 单股查询

```
茅台现在多少钱？

帮我看一下宁德时代的市盈率和换手率。
比亚迪今天涨了多少？涨了的话主力资金净流入是多少？
```

## 6. 复合策略（让 Claude 自动串工具）

```
今天 "AI 算力" 概念板块里主力净流入最多的 3 只票，分别给我看一下行情。
→ Claude 自动: sector_fund_flow → 取概念成分 → main_fund_rank → 取 top3 → get_stock_quote ×3

我想做强势股回调策略：找今天涨幅大于 5% 但 K 线 5 日均线没破的票。
→ Claude 自动: main_fund_rank 拿涨幅榜 → get_kline 逐个验证

帮我对比茅台和五粮液最近 30 天走势，谁更强？
→ Claude: get_kline ×2 → 算涨幅 → 输出对比

今天涨停板里主力资金最猛的 5 只，给我代码 + 板块。
```
